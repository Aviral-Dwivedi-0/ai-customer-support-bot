   └─ Result: https://ai-support-frontend.vercel.app

┌───────────────────────────────────────────────────────────────────────────┐
│                        6 SIMPLE STEPS                                     │
└───────────────────────────────────────────────────────────────────────────┘

STEP 1: Sign Up for Vercel
        → Go to vercel.com
        → Sign in with GitHub
        ⏱️ Time: 2 minutes

STEP 2: Deploy Backend
        → New Project → Set root to "backend"
        → Add GEMINI_API_KEY
        → Deploy & save backend URL
        ⏱️ Time: 5 minutes

STEP 3: Update Frontend Code
        → Edit frontend/src/App.js
        → Add backend URL
        → Git commit & push
        ⏱️ Time: 3 minutes

STEP 4: Deploy Frontend
        → New Project → Set root to "frontend"
        → Add REACT_APP_API_URL (backend URL)
        → Deploy & save frontend URL
        ⏱️ Time: 5 minutes

STEP 5: Update CORS
        → Edit backend/app/main.py
        → Add frontend URL to CORS
        → Git commit & push
        ⏱️ Time: 3 minutes

STEP 6: Test & Celebrate!
        → Open frontend URL
        → Test chatbot
        → Success! 🎉
        ⏱️ Time: 2 minutes

⏱️ TOTAL TIME: ~20 minutes
